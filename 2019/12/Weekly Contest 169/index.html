1307. Verbal Arithmetic Puzzle

描述花里胡哨，实际就是classic的Backtracking。

以"SEND" + "MORE" = "MONEY" 为例。

就等于1*D + 10*N + 100*E + 1000*S + 1*E + 10*R + 100*O + 1000*M - 1*Y - 10*E - 100*N - 1000*O -10000*M = 0

所以，只需要用一个boolean[] used来记录十个数字哪个用了哪个没用，用过的不能再用。List<Integer> list 和int[] count分别是用来记录有哪些字母出现了和每个字母需要的数量。

需要注意题目说了开头字母不能为0，所以还需要boolean[]nZero来标记哪几个字母不能映射成0。

全都搞定之后写个basic的backtracking就阔以了。
