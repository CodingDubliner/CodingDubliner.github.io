正则表达式

  • +：匹配前面的子表达式一次或多次（大于等于1次）。例如，”zo+“ 能匹配”zo”以及“zoo”，但不能匹配“z”。+等价于 {1,} 。

  • ？：匹配前面的子表达式零次或者一次。例如“do(es)?” 可以匹配”do“或者”does“。？等价于{0,1}。

常用正则表达式

  • String[] str=s.split(“ “); 分隔单个空格

  • String[] str=s.split(“\s+”); 分隔多个空格

  • String[] str=split(“\.”); 分隔 .

  • String[] str=split(“,”); 分隔，

  • String[] str=split(“/+”); 分隔/

String vs StringBuilder vs StringBuffer

StringBuilder

  • String：字符串常量，对象一旦创建之后该对象是不可更改

  • 对String 对象的任何改变都不影响到原来对象，相关的任何change操作都会生成新的对象

  • StringBuilder：对象是变量，可更改

  • StringBuilder 空间复杂度 O(n)

append() O(1)

reverse() O(n)

toString()O(n)

StringBuffer

  • StringBuilder是线程不安全的，而StringBuffer是线程安全的

  • StringBuffer很多方法可以带有synchronized关键字

解题思路

这一题是两道动态规划的嵌套。

我们要解决的大问题是：dp[i]代表从 $0$ 至 $i $最少切割多少刀。

然后我们可以从$0 ～ i$ 中切一刀 $j$，当substring $i ，j$是回文的时候，我们可以通过dp[j - 1] + 1得到最后的答案。
